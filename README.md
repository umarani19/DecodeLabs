# PROJECT 2: EXPENSE TRACKER
## Overview
The Expense Tracker is a simple Python project used to calculate daily expenses. Users can enter expense amounts, and the program adds them to show the total spent amount.
## Objective
The main objective of this project is to learn Python basics like user input,calculations,loops,and data processing.
## Tools Used
Python
VS Code
Terminal
## Conclusion
By this Expense Tracker project,I learned how to use Python for basic calculations and data handling.This project helped me understand concepts like loops,user input,and simple arithmetic operations.It also improved my coding,problem-solving,and logical thinking skills while building a useful daily expense management application.
